# ORIGIN NOTES

Project catalyst:
- Dissatisfaction with OpenClaw (AI slop, too large, too many dependencies).
- Friction building Slugger as a plugin inside a massive system.
- Realization: strong opinions about harness design.
- Upcoming Fix Hack Learn (FHL) at Microsoft created urgency.

Ari's motivation:
- This project is both work and personal.
- Work: multi-factor optimization.
- Personal: fun + learning.

Harness principles emerging:
- Simplicity over scale.
- Minimal dependencies.
- Stay close to the agent loop.
- Muscle memory must live in core execution path.
- Plugins are too far from the loop for deep behavioral shaping.
